### Instructions

Run this in a sheet editor. If you want to use the copy button, enable the macros

This file generates a script to ping using advanced options on Fortigate devices. Just provide the details you you need for the ping or trace or use the standard values within the "()", copy the result generated and paste on your SSH session with the Fortigate
